# Sample JAVA CLI App (with Gradle)

## Development

### Build and Run project

If you have gradle locally installed
''' shell
$ gradle build
$ java -jar build/libs/javabasics-1.0-SNAPSHOT.jar
'''

If you don't have gradle locally installed
'''shell
$ ./gradlew build
$ java -jar build/libs/javabasics-1.0-SNAPSHOT.jar
'''
